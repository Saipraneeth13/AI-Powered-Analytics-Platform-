## 📌 Overview

AI-Powered Analytics Platform is an intelligent data analytics solution that transforms raw datasets into meaningful insights using Artificial Intelligence and Machine Learning techniques. The platform provides interactive dashboards, predictive analytics, real-time visualizations, and automated reporting features.

It helps users analyze large datasets, discover patterns, forecast trends, and make data-driven decisions through an intuitive and modern interface.

# ⚙️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Saipraneeth13/AI-Powered-Analytics-Platform-.git
cd AI-Powered-Analytics-Platform-
```

---

## 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 3️⃣ Run Backend Server

```bash
python app.py
```

---

## 4️⃣ Start Frontend

```bash
cd frontend
npm install
npm start
```
